CATEGORY 4: EDUCATIONAL DATA (Students & Schools)
4.1- **Educational Institution & Performance (Columns AM - AR)**
  - SchoolID (AM): Link to active Educational Institution profile.
  - GradeYear (AN): e.g., Standard 7, Form 5, Year 1.
  - TeacherLead (AO): Supervising teacher or PE Head.
  - HouseColor (AP): School house (e.g., Red, Blue) for inter-house engagement.
  - NationalLeagueID (AQ): Association-level ID for competitive scouting.
  - AcademicAlert (AR): failing status -> triggers "Play Only if Pass" safeguards.

- **Connectivity & Ecosystem (Columns AS - AV)**
  - WhatsAppNumber (AS): Deep-linking connectivity for zero-rated WhatsApp.
  - FBPageLink (AT): Link to official Facebook business/association page.
  - GroupChatURL (AU): Invitation link for team WhatsApp/FB group.
  - PreFillMessage (AV): Message template for automated one-tap outreach.

- **Short-String Keys (Sync Optimization)**
  - cb (color blindness)
  - baha (hearing aid)
  - episodic (disability duration)
  - daily (training frequency)
  - moderate (training intensity)

- **Technical Sync Metadata**
  - Background Pulse: 15 minutes.
  - Packet Limit: 500KB.
  - Conflict Resolution: Last-Write-Wins (Timestamp-based).
School Type (Government, Private, Rural)
School Location (Village/City)

4.2 Academic Performance (Private - Guardian/School Only)

Current GPA / Academic Standing
Academic Alert Status (Boolean - Column AR: Grades low?)
Attendance Rate (%)
Academic Subjects (for scholarship matching)
Extra-Curricular Participation (clubs, societies)
Scholarship Status (Recipient, Applicant, None)

4.3 School Sports Representation
